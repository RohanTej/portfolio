Snake AI

Live Preview Source Code

Technologies Used

  • Python
  • PyGame

Overview

An algorithm whose goal is to get as large as possible in the game of snake.


What I learned

Defining a goal wasn't as easy as I expected. The snake would constantly run into itself if it determines that the path through itself is the fastest way to its 'food'. It was a learning experience to find out how specific we have to be when writing an AI algorithm.
