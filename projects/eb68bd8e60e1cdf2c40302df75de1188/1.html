Technologies Used

  • HTML
  • CSS + Bootstrap
  • JS + JQuery
  • PHP

Overview

As part of our internet programming course, we had to replicate an existing website. As a fun side project, I added a couple of more features to it.


Noteable Features

  • Signup and Login Features with expandable tier system
  • Google Maps integration (under 'Manage' page)
  • Weather forecast using Dark Sky API (under 'Manage' page)

What I learned

This was my first project in a very long time. At the time, for me this was a fairly large project and keeping track of multiple pages was quite a task. What made it worse was my unorganised coding style. This made me realise the importance of writing clean, understandalbe, and organised code. I've tried to apply these principles on all my subsequent projects.
